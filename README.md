GSM-Based-Fire-alert-System

## Project Scope
The GSM Based Fire Alert System is designed to detect fire using a flame sensor and immediately notify the user through a GSM module. The system also activates a buzzer and LED to provide a local alarm. This project improves safety by enabling quick response to fire incidents in homes,offices,industriesand public buildings.

## Project Desciption
This Proejct uses an Arduino,a flame sensor,a GSM module,a buzzer,and an LED. When the flame sensor detects fire,the microcontroller processes the signal,turns on the buzzer and LED,and sends an SMS alert to the registered mobile number through the GSM module.The system provides both local and remote fire notifications.
